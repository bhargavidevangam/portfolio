## Scripts

- `npm start` - Run development server
- `npm run build` - Create production build
- `npm test` - Run test suite
- `npm run deploy` - Deploy build via GitHub Pages

---

## Customization Guide

Update your profile content in the data layer:

- `src/data/headerData.js`
- `src/data/aboutData.js`
- `src/data/skillsData.js`
- `src/data/experienceData.js`
- `src/data/projectsData.js`
- `src/data/contactsData.js`
- `src/data/socialsData.js`
- `src/data/themeData.js`

Replace static assets in:

- `src/assets/png/`
- `src/assets/svg/`
- `src/assets/pdf/`
